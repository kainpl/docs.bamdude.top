## :material-lifebuoy: Отримання допомоги

### Ресурси для самостійного вирішення

1. **Пошук у документації** -- Використовуйте рядок пошуку
2. **Перевірте розділ усунення неполадок** -- Поширені проблеми описані
3. **Перегляньте Issues на GitHub** -- Можливо, хтось вже вирішив вашу проблему

### Підтримка спільноти

- [GitHub Issues](https://github.com/kainpl/bamdude/issues) -- Повідомляйте про помилки, пропонуйте функції
- [GitHub Discussions](https://github.com/kainpl/bamdude/discussions) -- Задавайте питання, діліться порадами
