# Трекер обслуговування

Плануйте та відстежуйте завдання з обслуговування, щоб ваші принтери працювали надійно. BamDude включає типи обслуговування з урахуванням моделі принтера та детальне журналювання історії.

---

## :material-tools: Огляд

Трекер обслуговування допомагає:

- **Планувати** регулярні завдання з обслуговування
- **Відстежувати** коли обслуговування було виконано останній раз
- **Отримувати сповіщення** коли настав час обслуговування
- **Вести журнал** детальної історії обслуговування з нотатками

---

## :material-format-list-checks: Типи обслуговування

### Типи за замовчуванням

BamDude включає завдання з обслуговування з урахуванням моделі принтера:

| Тип | Інтервал за замовчуванням | Застосовується до |
|-----|--------------------------|-------------------|
| **Очищення робочого столу** | Кожні 25 годин | Усі принтери |
| **Очищення сопла/хотенда** | Кожні 100 годин | Усі принтери |
| **Перевірка натягу ременів** | Кожні 200 годин | Усі принтери |
| **Перевірка PTFE трубки** | Кожні 500 годин | Усі принтери |
| **Змащування карбонових напрямних** | Кожні 50 годин | X1/P1 |
| **Очищення карбонових напрямних** | Кожні 100 годин | X1/P1 |
| **Змащування лінійних рейок** | Кожні 50 годин | A1/H2D |
| **Очищення лінійних рейок** | Кожні 100 годин | A1/H2D |

### Власні типи

Створюйте власні завдання з обслуговування у **Налаштування** > **Обслуговування**.

### Приховування типів за замовчуванням

Видаліть непотрібні типи за замовчуванням, натиснувши іконку видалення. Відновіть усі типи за замовчуванням кнопкою **Відновити завдання за замовчуванням**.

---

## :material-calendar-clock: Типи інтервалів

| Тип | Опис |
|-----|------|
| **Години друку** | На основі фактичного часу друку (наприклад, кожні 100 годин) |
| **Календарні дні** | На основі реального часу (наприклад, кожні 30 днів) |

---

## :material-check-circle: Журналювання обслуговування

1. Перейдіть на сторінку **Обслуговування**
2. Знайдіть завдання, що потребує виконання або прострочене
3. Натисніть **Позначити виконаним**
4. Додайте нотатки (що було зроблено, які деталі замінено)
5. Лічильник скидається

---

## :material-alert: Статус виконання

| Статус | Значення |
|:------:|----------|
| :material-check-circle:{ style="color: #4caf50" } OK | Ще не настав час |
| :material-alert:{ style="color: #ff9800" } Скоро | Наближається термін (80% інтервалу) |
| :material-alert-circle:{ style="color: #f44336" } Прострочено | Термін минув |

---

## :material-history: Історія обслуговування

Переглядайте детальну історію для кожного принтера:

| Поле | Опис |
|------|------|
| **Дата** | Коли виконано |
| **Тип** | Тип завдання з обслуговування |
| **Нотатки** | Що було зроблено |
| **Деталі** | Замінені деталі |

Експортуйте історію у CSV з вибором діапазону дат, для окремого принтера або всіх принтерів.

---

## :material-bell-ring: Сповіщення

Отримуйте сповіщення про необхідність обслуговування через будь-який налаштований провайдер сповіщень. Увімкніть подію **Обслуговування необхідне** у **Налаштування** > **Сповіщення**.

Користувачі Telegram-бота отримують інтерактивні сповіщення з вбудованою кнопкою **Позначити виконаним**.

---

## :material-lightbulb: Поради

!!! tip "Очищення столу"
    Протирайте ізопропіловим спиртом між друками для кращої адгезії. Раз на тиждень мийте засобом для посуду.

!!! tip "Документуйте все"
    Додавайте нотатки при журналюванні обслуговування, щоб накопичувати історію того, що працює.

!!! tip "Профілактика краща за ремонт"
    Дотримуйтесь інтервалів, не чекайте поки виникнуть проблеми. Профілактичне обслуговування запобігає дорогим поломкам.
